To suppress blank lines around selected text for Region template:

  1. Add DWORD registry value PadRegions

    • VS 2008:

      HKCU\Software\Microsoft\VisualStudio\9.0\DPack\SurroundWith

    • VS 2010:

      HKCU\Software\Microsoft\VisualStudio\10.0\DPack\SurroundWith

  2. Set value to 0

Or, use PowerShell to do it for you:

  1. Open PowerShell

  2. Paste in

    • VS 2008:

      Set-ItemProperty -Path HKCU:\Software\Microsoft\VisualStudio\9.0\DPack\SurroundWith -Name PadRegions -Type DWord -Value 0

    • VS 2010:

      Set-ItemProperty -Path HKCU:\Software\Microsoft\VisualStudio\10.0\DPack\SurroundWith -Name PadRegions -Type DWord -Value 0
