How to Add a Custom Search Provider in IE9 and Make it Default

To add a custom search provider:

  1. Open an IE9 web brower

  2. Enter http://www.microsoft.com/windows/ie/searchguide/en-en/default.mspx in the address bar

  3. If the provider you want is listed under Web Search, just click its name

  4. When you click on the link, a new page opens with a list of compatible search providers. Browse to the one you want to add (in this case, Google) and click on the button that says Add to Internet Explorer

Once you click on Add to Internet Explorer, another dialog box appears asking if you want to make it your default search provider. If you wish to add Google as search provider and make it default, select the option. Click Add and close the add-ons page.

Note: The new search provider will not appear in the Manage Add-ons window that is open. To view it, close and re-open it.
