If you're wanting to experiment with some dummy data, maybe you want to play around with some styles, or formatting. Or you might want to figure out how to use outlining. In both cases, you don't want to have to type all of the text yourself. Well, you can use this little trick instead

In a blank document (or anywhere in an existing document if you want), type:

    =rand(x) and press enter (where x is the nunber of paragraphs of text that you want)

If you'd typed:

    =rand(10) and pressed enter

You'd now have 10 paragraphs of random text (just like I ened up when I tried it in Word).
